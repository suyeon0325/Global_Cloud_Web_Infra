**AWS Multi-Region Web Service Architecture & Infra**


🖥️ 프로젝트 소개
AWS 클라우드포메이션을 활용한, 글로벌 리전 웹 인프라 구현
- 고가용성 웹 서비스


📌 주요 기능
- 웹트래픽 최적화
- 고가용성 장애조치
- Cloud <-> 온프레미스 영역간 Private 통신 제공
- Private DNS 쿼리 해결 


🕰️ 개발 기간
2024.01.24 - 2024.02.03


⚙️ 이용한 AWS 서비스
- AWS EC2 Instance
- ALB (Application Load)
- Route53
- Site to Site VPN
- TGW (Transit Gateway)
- GA (Global Accelerator)


🛠 개발 환경
- Database : MariaDB
